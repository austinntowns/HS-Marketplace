   - Export `'use server'` at top (Server Action)
   - Define `ListingFilters` interface:
     ```typescript
     interface ListingFilters {
       types?: string[]
       states?: string[]
       minPrice?: number
       maxPrice?: number
       cursor?: string  // ISO timestamp from createdAt
       sort?: 'newest' | 'price-asc' | 'price-desc'
     }
     ```
   - Define `ListingCard` return type:
     ```typescript
     interface ListingCard {
       id: string
       type: 'suite' | 'flagship' | 'territory' | 'bundle'
       locationName: string | null
       city: string | null
       state: string | null
       askingPrice: number | null
       latitude: string | null
       longitude: string | null
       createdAt: Date
       primaryPhotoUrl: string | null
     }
     ```
   - Implement `getListings(filters: ListingFilters)`:
     - PAGE_SIZE = 12
     - Build conditions array:
       - `eq(listings.status, 'active')` always
       - `inArray(listings.type, filters.types)` if types provided
       - `inArray(listings.state, filters.states)` if states provided
       - `gte(listings.askingPrice, filters.minPrice)` if minPrice provided
       - `lte(listings.askingPrice, filters.maxPrice)` if maxPrice provided
       - `lt(listings.createdAt, new Date(filters.cursor))` if cursor provided
     - Order by:
       - 'newest': `desc(listings.createdAt)` (default)
       - 'price-asc': `asc(listings.askingPrice)`
       - 'price-desc': `desc(listings.askingPrice)`
     - Limit to PAGE_SIZE + 1 (extra to detect hasMore)
     - Left join photos table, select first photo (order = 0) as primaryPhotoUrl
     - Return `{ items: ListingCard[], nextCursor: string | null }`
